關西自由行導覽手冊

👗 4月季節穿著建議

4月大阪氣溫約 10°C - 20°C，正值櫻花末期/新綠期。

  • 洋蔥式穿法：早晚偏涼，白天太陽下舒適。
  • 建議：薄長袖/襯衫 + 防風外套或薄風衣。若要去京都山區(清水寺)，多備一件輕便發熱衣。

🛫 機場流程與快速通關

1. 出發前準備 (快速通關)
  • 註冊 Visit Japan Web (VJW)，填寫「入境審查」與「海關申報」。
  • 截圖 QR Code，下機後依照指示走「電子申報通道」可極速入境。
2. 托運前流程
  • 檢查行動電源(不可托運，須隨身)。
  • 液體需裝入 20x20cm 透明夾鏈袋 (單瓶 < 100ml)。

🎫 KKDAY 票券兌換詳細說明

A. HARUKA 關西機場特快
  • 地點：關西機場車站 (JR 綠色售票機)。
  • 方式：掃描 KKday 電子 QR Code + 掃描護照 → 領取實體車票。
  • 搭乘：持票進站，HARUKA 為全車對號座(需於機器劃位)或自由席(4-6車廂)。
B. 大阪城天守閣
  • 方式：直接出示 KKday 電子憑證 QR Code 掃描進場，無需換票。
  • 注意：若排隊人數多，請確認「已持有票券」通道，通常可快速排隊進入。

💳 ICOCA 卡片購買與儲值

  • 購買：JR 關西空港站「自動售票機」(藍色機器)。選擇「購入ICOCA」，初次購買通常為 2,000円 (含 500円押金 + 1,500円儲值金)。
  • 儲值方式： 1. 機器：插入卡片 → 點選「チャージ(Charge)」→ 投入現金。 2. 超商：持卡跟店員說「Charge」並出示現金即可。
