👗 4月季節穿著與住處

住處：大阪府大阪市中央區南本町4-2-9 (+81 6-6258-8911)

  • 氣溫：10°C - 20°C。早晚溫差極大。
  • 建議：薄長袖為主，外加一件防風保暖外套。4月風大時體感偏冷，建議備妥圍巾或輕便發熱衣。

🛫 機場通關與行李流程

1. 出發前準備：快速通關
  • 註冊 Visit Japan Web (VJW)，完成「入境審查」及「海關申報」。
  • 截圖：一定要將產出的 QR Code 截圖存檔，下機後依照藍色指示牌走「電子申報通道」。
2. 托運前流程
  • 行動電源：嚴禁托運，必須隨身攜帶。
  • 液體：手提單瓶 < 100ml，需裝入20x20cm透明夾鏈袋。托運單瓶 < 500ml。
  • 電捲棒：若電池不可拆卸，日本海關嚴禁攜帶（不論托運或隨身）。

🎫 KKDAY 票券兌換方式

A. HARUKA 關西機場特快 (來回票)
  • 地點：機場車站「JR 綠色售票機」。
  • 操作：點選螢幕「QR Code 兌換」→ 掃描 KKday 憑證 → 掃描護照 → 領取實體票。
  • 劃位：領到實體票後，可在機器點選「指定席預約」劃位，或直接搭乘 4-6 車廂的自由席。
B. 大阪城天守閣
  • 兌換：無需去櫃檯排隊。直接在天守閣入口出示手機 KKday QR Code，由工作人員掃描後直接進入。

💳 ICOCA 購買與儲值操作

購買位置：機場 JR 站藍色機器
  • 機台上有「購入 ICOCA」按鈕。
  • 費用：2,000円（內含 500円押金 + 1,500円可用餘額）。
儲值方式 (Charge)
  • 機台：將卡片放入，選擇「チャージ(Charge)」，投入現金（1,000円起）。
  • 超商：持卡給店員看，說「Charge (恰吉)」，並給現金即可。
